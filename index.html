Geocode missing pins

For country-level Relais pins, this will try OpenStreetMap Nominatim one by one. Use sparingly and export after. If blocked, use Google Maps links per card.

Nominatim has usage limits. For production, use a paid geocoder or run the Python updater included next to this HTML.

Suggested updater workflow

1. Open official sources: Aman hotels, Belmond hotels, Relais sitemap.\n2. Run the included Python updater with Playwright/BeautifulSoup.\n3. Review hotel vs restaurant classification.\n4. Paste/export JSON into this page.\n5. Verify seasonal direct NYC flights before booking.
